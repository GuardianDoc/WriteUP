www-data​

上传Shell 后我开始访问站点

www-data —> root (第一种方式)​

经过枚举, 发现 find 指令存在 SUID 特权

www-data —> c0ldd (第二种方式)​

在网站的配置文件中发现了 c0ldd 的密码

尝试切换用户发现密码正确

c0ldd —> root (第二种方式)​
